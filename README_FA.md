# 🎧 پخش‌کننده موسیقی Noctra

یک موزیک پلیر ساده و سبک تحت وب ساخته شده با HTML، CSS و JavaScript.

## ✨ ویژگی‌ها
- 🎵 پخش / توقف موسیقی
- ⏭️ رفتن به آهنگ بعدی / قبلی
- 🔊 کنترل صدا
- 📱 طراحی کاملاً ریسپانسیو
- 🎨 رابط کاربری مدرن و تمیز
- 📂 امکان اضافه کردن آهنگ‌های شخصی
