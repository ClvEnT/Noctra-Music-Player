# 🎧 Noctra Music Player

A sleek and minimal web-based music player built with HTML, CSS, and JavaScript.

## ✨ Features
- 🎵 Play / Pause music
- ⏭️ Next / Previous track
- 🔊 Volume control
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- 📂 Easy to add your own music files

## 🚀 Demo
Just open `index.html` in your browser and enjoy.
